## Tujuan

Menerapkan pemahaman tentang kelas String dan kelas File (I/O) dalam Java untuk membangun sistem sederhana yaitu perpustakaan SMP yang menangani data siswa, buku, pegawai serta transaksi peminjaman dan pengembalian buku


## Deskripsi Soal

Sebuah SMP mempunyai perpustakaan yag mempunyai catatan data siswa, buku, jenis buku dan pegawai. Perpustakaan tersebut melayani kegiatan peminjaman dan pengembalian buku. Buku yang dipinjam maksimum sebanyak 2 buku. Anda diminta membuat program perpustakaan dengan kemampuan sbb:
1. Program mempunyai menu untuk dapat mengisi, mengedit, menghapus dan melihat data siswa, buku dan pegawai. Data-data tersebut tersimpan di file (ekstensi .txt)
2. Program hanya bisa dijalankan oleh pegawai. Untuk itu jika program dijalankan maka pegawai harus login
3. Data siswa adalah NIS, Nama, Alamat. Data buku adalah kode, judul, jenis buku. Adapun pegawai adalah NIP, Nama dan Tanggal lahir.
4. Data peminjaman terdiri atas kode transaksi, NIS, Kode Buku, Tanggal Pinjam, Tanggal Pengembalian dan Status (0: belum dikembalikan, 1:dikembalikan)
5. Program bisa melihat laporan buku apa saja yang belum dikembalikan dan siapa yang meminjam melebihi jatuh tempo. Untuk laporan lainnya silahkan berkreasi
6. Berikan penanganan exception untuk operasi file
7. Buat menu untuk memilih aktifitas (input data, transaksi maupun melihat laporan) perpustakaan.
8. Yang dikumpulkan adalah kode program, screenshot output program untuk masing-msing menu (feature). Berkas pengumpulan adalah file pdf.
